1. Product Overview
1.1 Problem Statement
Employees experiencing IT issues currently rely on logging tickets and waiting for Service Desk (SD) response — a process that is time-consuming and disrupts productivity. Many issues are repetitive and resolvable with guided self-service. There is no on-site, intelligent, walk-up IT support available to employees.

1.2 Product Vision
TED (Tech Express Desk) is an AI-powered, self-service IT support kiosk installed on-site at office locations. Employees walk up to TED, connect their device, and receive instant AI-driven diagnostics, guided troubleshooting, and resolution — reducing reliance on the Service Desk for common IT issues.

1.3 Product Goals
Reduce average IT ticket resolution time by enabling instant self-service diagnostics
Deflect common/repeat IT issues from the Service Desk queue
Provide AI-assisted, step-by-step guided resolution to employees
Auto-create Freshworks tickets with pre-populated diagnostic data for unresolved issues
Enable loaner device dispensing from kiosk locker for hardware failures

3. Core Application Flow
3.1 Happy Path — Self-Service Resolution
Employee walks up to TED kiosk at office location
Employee authenticates (badge tap / SSO login)
Employee connects device to TED via USB or selects issue type on screen
TED System Engine runs automated hardware + software diagnostics
Issue detected → AI Engine activates and attempts auto-fix
Fix applied → Employee confirms resolution → Session ends

3.2 Escalation Path — SD Ticket Creation
Issue not detected or not resolved by System Engine
AI Engine analyses error via screenshot / screen recording
AI diagnoses error, analyses past tickets and KB articles
AI provides step-by-step fix recommendation to employee
Employee attempts fix with AI guidance
Still unresolved → TED auto-creates Freshworks ticket
Ticket pre-populated with: diagnostics data, error screenshots, AI analysis, suggested assignment group
Ticket routed to SD or AI-suggested assignment group (with SD validation before assignment)
SD resolves issue using AI suggestion, KB, and past resolved tickets

3.3 Loaner Device Path
Hardware failure confirmed by diagnostic engine
TED dispenses loaner device from integrated locker unit
Loaner request auto-logged in Freshworks
Employee collects device and continues work

4. AI Diagnostic Engine
4.1 Capabilities
Hardware diagnostics: CPU, RAM, storage, battery, connectivity
Software diagnostics: OS errors, driver issues, application crashes
Visual error analysis via screenshot or screen recording capture
RAG-based lookup: past resolved tickets + KB articles
Step-by-step fix generation with confidence scoring
Smart ticket routing: AI-suggested assignment group based on issue type
